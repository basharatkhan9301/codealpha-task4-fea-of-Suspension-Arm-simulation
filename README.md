# FEA Simulation of Suspension Arm of Four Wheeler

## Overview
Finite Element Analysis (FEA) simulation performed on a Suspension Arm as part of the CodeAlpha Mechanical Design Internship (Task 4).

## Tools Used
- PTC Creo Parametric (model preparation)
- SimScale (simulation)

## Contents
- `simulation_files/` — Model export files and simulation setup
- `images/` — Result plots (stress, displacement, etc.)
- `report/` — Task report (PDF/Word)

## Description
A mechanical component was modeled and prepared in Creo, then analyzed using FEA to evaluate stress distribution and deformation under applied loading conditions.
